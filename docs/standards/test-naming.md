# Convenção de Nomenclatura de Testes

Os testes automatizados devem seguir estritamente o padrão de Given-When-Then no descritivo (blocos `it` ou `test`).
- Exemplo: `it('should return 401 unauthorized when user is not authenticated')`
- O `describe` deve agrupar o contexto em blocos claros (ex: nome da função ou caso de uso).
